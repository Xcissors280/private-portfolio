# Derek Buchwald - Professional Portfolio

A professional online portfolio showcasing my experience in engineering, CAD design, robotics, and community service.

## Features

- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Blue/Turquoise Color Palette**: Professional and modern aesthetic
- **Comprehensive Sections**:
  - Home with engaging hero section
  - About section with education and certifications
  - Projects featuring robotics CAD, carpentry work, and technical coursework
  - Leadership and experience timeline
  - Community involvement and volunteer work
  - Contact information

- **OSHA 10-Hour**: Workplace safety certification
- **Leadership**: Team management, project coordination
- **Technical**: CAD design, manufacturing, engineering

## Projects

## Getting Started

1. Clone or download this repository
2. Customize the images folder with your portfolio photos
3. Update the `resume.pdf` file with your resume
4. Deploy to GitHub Pages or your preferred hosting

## Customization

- Edit `index.html` to update content
- Modify `styles.css` to change colors and styling
- Add images to the `images/` folder
- Update contact information as needed
